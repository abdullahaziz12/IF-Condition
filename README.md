# 🔀 C++ Conditional Statements — Practice Repository

A collection of small, independent C++ programs written while practicing **conditional statements** (`if`, `else if`, `else`) and decision-making logic. Each file focuses on a single problem, covering number classification, comparisons, and simple rule-based systems.

## 📚 About

This is not a single project — it's a growing set of standalone `.cpp` exercises, each demonstrating conditional logic: comparisons, nested conditions, and multi-branch decision-making.

## 📁 What's Inside

**Number Classification**
- Even & Odd Number
- Positive & Negative Number
- Positive, Zero, Negative
- Divisible Number
- Equal Numbers

**Comparisons**
- Greater Number Finder
- Smallest and Largest Number Finder
- Multiplication of Two Numbers

**Geometry & Date Logic**
- Equilateral, Isosceles, Scalene Triangle Finder
- Leap Year Finder

**Real-World Simulations**
- Salary Deductor
- Admission System
- Categorize People According to Age

**Character Logic**
- Vowel & Non-Vowel Checker

Each `.cpp` file is self-contained and can be compiled and run independently. Pre-compiled `.exe` files are included for quick testing on Windows.

## 🎯 Purpose

- Practice writing conditional logic (`if`, `else if`, `else`) in C++
- Build comfort with comparison operators and multi-branch decisions
- Strengthen problem-solving through small, real-world-inspired scenarios
- Create a personal reference library of common conditional patterns

## 🚀 Technologies Used

- C++
- Dev C++ / g++ compiler

## 🛠️ How to Run

```bash
g++ "filename.cpp" -o output
./output
```

Replace `filename.cpp` with the program you want to run. Each file is independent — no shared dependencies between them.

## 📌 Note

This repository is a practice log, not a finished application. It's continuously updated as new conditional-logic exercises are solved.
